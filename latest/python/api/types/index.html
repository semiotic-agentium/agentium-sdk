    Chain Identifiers¶

    agentium_sdk.Caip2 dataclass ¶

    CAIP-2 chain identifier (e.g., "eip155:84532", "eip155:1").

    CAIP-2 defines a format for blockchain identifiers: namespace:reference - Namespace: 3-8 lowercase alphanumeric characters (e.g., "eip155", "solana") - Reference: 1-32 alphanumeric characters with hyphens/underscores

    See: https://github.com/ChainAgnostic/CAIPs/blob/master/CAIPs/caip-2.md

    Example

    caip2 = Caip2.parse("eip155:84532") caip2.namespace 'eip155' caip2.reference '84532' caip2.evm_chain_id() 84532

    namespace instance-attribute ¶

    namespace: str

    Chain namespace (e.g., "eip155", "solana", "cosmos").

    reference instance-attribute ¶

    reference: str

    Chain reference (e.g., "1", "84532", "cosmoshub-4").
